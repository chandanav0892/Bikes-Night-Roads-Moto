## Setup Instructions

### 1. Clone the repository

```bash
git clone https://github.com/chandanav0892/Bikes-Night-Roads-Moto.git
cd Bikes-Night-Roads-Moto
```
### 2. Create a virtual environment

>python3 -m venv .venv

For windows:
>.venv\Scripts\activate

For mac/linux:
>source .venv/bin/activate

### 3. Install required packages

>pip install -r requirements.txt

### 4. Create a .env file in the root folder of the project.

API_NINJAS_KEY=your_api_ninjas_key_here

API_NINJAS_URL=https://api.api-ninjas.com/v1/motorcycles

### 5. Run the Flask application

>python app.py

### 6. Open the application in browser

Open this URL: http://127.0.0.1:5000
